## Quick Start

1.  **Build and Run:**
    Navigate to the project root (where `docker-compose.yml` is) and run:
    ```bash
    docker-compose up --build
    ```
    The server will start on port `8084`.


2.  **Test**
    In your browser or with `curl -OJ`: `http://localhost:5000/download/wireguard`
    In your browser or with `curl`: `http://localhost:5000/list-files`
